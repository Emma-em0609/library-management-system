# Система управління бібліотекою

Курсова робота з дисципліни «Бази даних».  
Веб-застосунок для управління бібліотечним фондом: книги, читачі та облік видачі.

## Технології

- **Backend:** Node.js (http)
- **База даних:** PostgreSQL
- **Frontend:** HTML, CSS, JavaScript

## Функціонал

- Додавання, редагування та видалення книг
- Управління читачами
- Облік видачі та повернення книг
- Перевірка наявності книги перед видачею

## Запуск

1. Встанови залежності:
   npm install

2. Створи базу даних PostgreSQL з назвою `library`

3. Запусти сервер:
   npm start

4. Відкрий браузер: http://localhost:3000
